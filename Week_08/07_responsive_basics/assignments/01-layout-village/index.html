Responsive Layout Basics

Responsive flexbow layout using single media query.

Tablet (768px+) the layout is a row, mobile and tablet devices will see a column. Flipping the flex-direction property's value from its default (row) to column on mobile is all you have to do. And remember - flexbox natively takes over the width of children, so they are natively responsive.

Flexbox sidebar layout using single media query

Sidebars generally require more space to display properly, so this media query is at 960px. We just stack the sidebar below the main content, which is a best practice but really isn't very good IRL. This is one of the reasons sidebars have all but disappeared from modern web design.

Responsive CSS grid layout using cascading media queries

With CSS Grid, our approach is a little different. The fractional unit property (fr) is natively responsive so we don't have to worry about column widths. And remember the implicit grid - rows are getting created automatically, so we don't have to worry about that either. All we do is switch grid-template-columns at the media query break points. In this example we have multiple media queries making ranges - mobile (under 480px), phablet/tablet (480px-768px), desktop (769px - 1200px) and wide desktop (1200px+).

Using display: inline-block; (natively responsive)

This is dirty but it works. You can use inline-block declarations on the children and they will natively drop to new lines as the screen gets smaller. You don't have to use media queries which gives you less control over what's happening. This can be useful for aligning small icons but little else.

Responsive CSS grid layout using cascading media queries and ems

The fr unit makes ems mostly obsolete for grids but you can use it with grid-gap. The idea is that you can just adjust the font-size instead of the grid-gap itself in this example. How useful is this? Potentially somewhat useful if you've got exotic padding that needs to scale down to other devices. I find it maddening to link font-sizes and padding together but some people swear by this method.
